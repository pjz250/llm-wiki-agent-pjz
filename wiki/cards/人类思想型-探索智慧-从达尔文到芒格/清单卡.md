## 清单卡：误判心理学（28种心理倾向检查清单）

- **条目列表**：
  - [ ] 避免“联结效应”：根据事物本身的价值和功过来评价，而非其关联的愉快或痛苦经验。
  - [ ] 警惕“奖赏与惩罚”的力量：明确认可并奖励你期望的行为，而不是其他行为。
  - [ ] 警惕“利己倾向”：清晰识别决策背后各方的利益动机和激励机制。
  - [ ] 避免“自我服务偏见”：坦诚评估自己的优势和局限，避免过度自信。
  - [ ] 警惕“自我欺骗”与“否认”：面对现实，尤其是令人不悦的现实，而非歪曲事实以求心安。
  - [ ] 避免“一致性倾向”：勇于承认错误并根据新证据改变观点，主动寻找并验证与自己观点相悖的证据。
  - [ ] 警惕“剥夺性综合症”：基于当前和未来的价值做决策，而非沉没成本或历史成本。
  - [ ] 避免“满足现状”与“无为综合症”：意识到不作为本身就是一种决策，且可能产生代价。
  - [ ] 警惕“急功近利”：权衡决策的短期与长期后果。
  - [ ] 警惕“忌妒”：将注意力集中在自我成长和既定目标上。
  - [ ] 避免“对比失真”：基于事物的绝对价值进行判断，而非其与参照物的对比。
  - [ ] 避免“固锚效应”：从零基出发思考问题，避免锚定于初始信息，并根据现实情况调整参考点。
  - [ ] 警惕“生动性”和“时近效应”：区分信息的戏剧性/易得性与统计显著性/常规概率。
  - [ ] 避免忽略“遗漏信息”或抽象信息：系统性地寻找缺失信息、沉默证据和替代解释。
  - [ ] 警惕“互惠原则”：识别他人先给予后索取意图的策略。
  - [ ] 避免“喜好”过度影响：基于事实和逻辑做判断，而非对他人喜好的回应。
  - [ ] 避免“社交认证”偏见：在明确自己想法前，审慎对待多数人的意见和同行压力。
  - [ ] 避免“权威”过度影响：以证据而非头衔或声望评判观点的正确性。
  - [ ] 警惕“意义建构”的误导：寻找事件的多种解释，并将超自然的巧合视为概率下的大数法则。
  - [ ] 警惕“尊重理由”：对给出的、哪怕是空洞的理由，进行独立思考。
  - [ ] 警惕“先信后疑”：在重要问题上，谨慎对待第一印象，延缓对未经审慎信息的评价。
  - [ ] 警惕“记忆的局限性”：重要信息应记录在案，而非依赖记忆；警惕暗示对记忆的歪曲。
  - [ ] 避免“盲目行动”：在行动前问清行动的根本目的。
  - [ ] 避免“表态综合症”：若无真知灼见，勇于承认“我不知道”。
  - [ ] 警惕“情绪干扰”：在强烈的正面或负面情绪下，推迟重大决策。
  - [ ] 警惕“压力”的影响：以“解决问题”而非“担忧”的态度应对压力，提升掌控感。
  - [ ] 警惕“痛苦、化学品或疾病”：在身体和精神状况不佳时，避免做出重大决定。
  - [ ] 警惕“多种倾向合奏”：当多个心理倾向共同作用时，其结果往往是非线性的，需格外警惕。

## 清单卡：误判物理学和误判数学检查清单

- **条目列表**：
  - [ ] 进行“系统思考”：追踪行动的直接结果、次级结果及长期后果，而非孤立看待单一事件。
  - [ ] 考虑“尺寸与极限”：评估方案时，考虑规模、时间、临界点和系统中最薄弱的环节。
  - [ ] 正确地理解“因果逻辑”：重大结果可能源于微小原因；警惕以相关关系直接充当因果关系；考虑多重原因并对比参照组。
  - [ ] 量化考量“数字与意义”：使用基本的数学进行量化计算，区分绝对风险和相对风险；认识指数增长的威力；理解金钱的时间价值。
  - [ ] 理性对待“概率与风险”：同时关注事件发生的概率与结果的严重性；理解独立事件无记忆；区分期望价值与一次性赌博的效用。
  - [ ] 审慎评估“方案”：将复杂方案分解为关键步骤，并估算关键路径的成功概率；考虑系统所有可能的故障模式。
  - [ ] 预留“安全边际”：在决策中主动留出缓冲空间以吸收不确定性，应对未知风险。
  - [ ] 运用“逆向思维”：从“如何导致失败”出发，制定预防方案；从目标倒推必要条件。
  - [ ] 整合并使用“多学科模型”：处理复杂问题时，从数学、物理、生物、心理、经济等多学科视角进行评估。
  - [ ] 科学地评估“证据”信度：在引入新证据时，将其与基准比率（先验概率）相结合；评估假阳性与假阴性的概率。
  - [ ] 警惕“非典型性证据”：评估过去的表现时，考虑运气成分、幸存者偏差、样本规模大小以及“回归平均值”效应。

## 清单卡：思维能力训练指南（12种思维方法）

- **条目列表**：
  - [ ] **基于现实的模型**：建立并持续更新由各重要学科的基本真理组成的思维模型格栅。
  - [ ] **理解“意义”**：务必理解概念、数字或陈述的真实含义，而非仅记住其名称；能用简单的语言解释清楚。
  - [ ] **简化**：区分主要矛盾与次要矛盾，并将复杂问题分解为基本要素。
  - [ ] **制定规则和过滤器**：制定个人化的决策标准和过滤器，以快速排除不重要的或危险的选项。
  - [ ] **设定清晰“目标”**：明确具体、可衡量且与最终需求一致的目标；明确达成目标的因果链。
  - [ ] **评估“多项选择”**：评估所有选项时，引入机会成本概念，并与其他最佳替代方案进行比较。
  - [ ] **追踪“结果”**：全面预测每个选项带来的直接与间接、短期与长期后果，包括对系统整体的影响。
  - [ ] **量化思考**：用基本数学知识评估影响，区分“粗略的正确”与“精确的错误”，不仅仅依赖可量化的数据。
  - [ ] **寻找“证据”**：使用科学方法和实验来检验假设，主动寻找并重视证伪的证据。
  - [ ] **运用“逆向思维”**：在解决问题时，思考“如何实现我们最想避免的结果”，避开错误以接近正确。
  - [ ] **管理“风险”**：系统性评估不确定性的来源和潜在负面影响，并强化下行风险管理。
  - [ ] **培养正确的“态度”**：保持诚实、客观、好奇心和终身学习的态度；活在当下并合理管理期望。

## 清单卡：通用决策检查清单（附录4核心提炼）

- **条目列表**：
  - [ ] **定义问题**：问题的实质、核心及关键变量是什么？是重要且可知的吗？
  - [ ] **理解含义**：我真的理解这个概念、数字或陈述的含义及其背后的因果关系吗？
  - [ ] **明确目标**：我最想实现（或避免）的具体结果是什么？这背后的原因是什么？
  - [ ] **运用规则与过滤器**：该决策是否符合我的基本筛选标准和价值观？
  - [ ] **评估替代选择**：与其他可选的方案相比，这个方案在机会成本、风险与回报上如何？
  - [ ] **预测结果**：这个行动的直接、间接及长期结果是什么？对整个系统会产生什么影响？
  - [ ] **检验假设与偏见**：我的判断是否受到某种心理倾向（如利己、一致性）的影响？我如何证明我的假设是错误的？
  - [ ] **评估不确定性**：哪些因素是未知的或不确定的？如果犯错，最坏的结果是什么，我能承受吗？
  - [ ] **量化价值**：该决策的期望价值是多少？与我的基准（如无风险利率）相比如何？
  - [ ] **制定实施计划**：谁、做什么、何时、何地、为什么、怎么做？我设置了监控进展的指标吗？
  - [ ] **进行事后剖析**：无论结果如何，分析成功或失败的根本原因，并从中学习。
