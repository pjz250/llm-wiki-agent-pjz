# 模型卡：自然选择与进化论框架

- **底层学科**：生物学、进化论
- **输入**：生物个体在特定环境中产生的随机基因突变；环境中的资源约束、天敌、疾病等选择压力。
- **输出**：预测那些能够提高个体生存与繁衍机会的特征（解剖、生理、行为）会倾向于被保留并遗传给后代；特征是否被选择取决于环境是否变化。
- **失效条件**：当环境变化速度快于生物适应速度时，原本有利的特征可能失效；进化解释无法直接应用于文化或社会层面的快速变迁；无法用进化论解释单个个体的短期行为选择（补）。

# 模型卡：痛苦-快乐原则

- **底层学科**：心理学、生物学
- **输入**：个体的行为选择；行为结果带来的痛苦或快乐感受。
- **输出**：人类会趋利避害，倾向于重复带来快乐/奖励的行为，避免导致痛苦/惩罚的行为。
- **失效条件**：当个体对长期痛苦或快乐的认知出现偏差时（如短视），该原则可能被扭曲；当行为受到强烈的社会规范、意识形态或心理疾病影响时，痛苦-快乐的短期计算可能失灵（补）。

# 模型卡：损失厌恶

- **底层学科**：心理学、行为经济学
- **输入**：个体对获得某物（如金钱、地位、物品）与失去同等价值某物的主观感受。
- **输出**：预测人类对失去的痛苦感受远大于对获得的快乐感受（约2倍），这导致个体在面对风险选择时倾向于保守，以避免损失。
- **失效条件**：当个体拥有非常充裕的资源或极端的财富水平时，损失厌恶的程度可能减弱；当面临生死攸关或极具诱惑力的巨大收益时，可能会暂时压倒损失厌恶倾向（补）。

# 模型卡：认知吝啬鬼/启发法-偏差模型

- **底层学科**：心理学、认知科学
- **输入**：需要解决的复杂问题或需要做的判断；大脑的有限认知资源（时间、注意力、计算能力）。
- **输出**：大脑倾向于优先使用快速、省力的直觉判断（启发法），而非费力的理性分析。这导致系统性的认知偏差（如过度乐观、从众、轻信权威等）。
- **失效条件**：当问题简单、时间充裕、认知资源充足且个体有强烈动机进行准确判断时，理性思考可能占上风；在高度专业化的领域（如科学实验），可通过严格的程序化思考来克服。

# 模型卡：误判心理学清单（28种心理倾向模型）

这是一个组合模型，包含28个具体的子模型。每个子模型都遵循相同的因果逻辑：人类大脑的固有倾向 → 在特定情境下 → 导致系统性的判断错误。这里列出关键的几个子模型：

## 模型卡：联结效应模型

- **底层学科**：心理学（条件反射）
- **输入**：某刺激（人、物、情境、声音）与过去的痛苦或快乐经历同时出现。
- **输出**：该刺激会被自动赋予与过去经历相同的痛苦或快乐情感，从而影响对它的判断，甚至导致误判。
- **失效条件**：当个体有意识地识别并分析这个自动联结，并根据事物本身的功过而非历史联想进行评价时，该效应可被削弱。

## 模型卡：奖励与惩罚模型

- **底层学科**：心理学、行为经济学
- **输入**：一个行为；该行为所带来的直接或间接的奖赏或惩罚结果。
- **输出**：预测个体倾向于重复能带来奖赏（快乐、利益）的行为，避免会导致惩罚（痛苦、损失）的行为；奖惩的即时性和确定性越强，影响力越大。
- **失效条件**：当奖励与长期目标相悖时（如奖金激励短期行为损害长期价值）；当惩罚的严重性与错误的后果不匹配时；当个体对“奖励”的认知存在偏差时。

## 模型卡：自我服务偏见模型

- **底层学科**：心理学
- **输入**：成功或失败的结果（尤其是带有运气成分的结果）。
- **输出**：个体倾向于将成功归因于自己的能力和努力，将失败归因于外部因素（运气、他人、环境），从而高估自己的能力，低估运气的作用。
- **失效条件**：当个体处于严重抑郁或自我批判的状态时；当失败带来的后果极为严重且无可辩驳时（补）。

## 模型卡：一贯性倾向模型

- **底层学科**：心理学
- **输入**：个体公开或私下做出的承诺（言语、选择、投资的行为）；与该承诺相矛盾的新的证据或环境。
- **输出**：个体倾向于为了维持与之前承诺、观点或形象的一致，即便面对相反证据，也会坚持原有决定，或为不合理行为寻找理由，甚至加倍投入（沉没成本谬误）。
- **失效条件**：当个体拥有极高的自我批判精神和认知灵活性时；当改变承诺所带来的社会/个人损失小于维持错误承诺的损失时。

## 模型卡：剥夺性综合症模型

- **底层学科**：心理学
- **输入**：个体已经拥有或即将到手（拥有所有权或控制权）的某样有价值的东西（物品、权力、自由、机会）面临被剥夺或失去的风险。
- **输出**：个体对该东西的价值评价会急剧上升，并产生强烈的反抗情绪和行动（如囤积、争抢、不惜代价挽回），往往导致非理性的决定。
- **失效条件**：当个体对失去的东西有非常清晰、冷静的成本-收益分析时；当东西本身对个体不再具备价值时。

## 模型卡：对比失真模型

- **底层学科**：心理学
- **输入**：对某个事物（价格、温度、大小、舒适度）的判断依赖于其与先前或同时出现的参考物（参照点）的对比，而非其绝对价值。
- **输出**：导致感知和判断的扭曲，例如：先看高价商品再看低价商品会觉得后者很便宜；忽略渐进式的微小变化。
- **失效条件**：当个体能够刻意抽离参照物，对事物进行独立评估时；当参考物与目标物差异巨大，无法形成有效比较时（补）。

## 模型卡：固锚效应模型

- **底层学科**：心理学、行为经济学
- **输入**：一个初始的、随机的、或由他人提供的参考值（锚点）。
- **输出**：在后续的判断和决策中，个体的估计会不自觉地、过分地向该“锚点”靠拢，导致最终结果被锚点扭曲。
- **失效条件**：当个体拥有丰富且相关的专业知识，能迅速识别并忽略不相关的锚；当锚定信息明显荒谬到被决策者直接拒绝时。

## 模型卡：社交认证/从众模型

- **底层学科**：心理学、社会学
- **输入**：在不确定、模糊或陌生的情境中，观察周围其他人的行为、观点或信念，尤其是与自己类似的人的行为。
- **输出**：个体倾向于认为大多数人的行为是正确的，并模仿跟随，即便这种行为有时是错误的（如群体疯狂、多元无知）。
- **失效条件**：当个体拥有强烈自信、独立思考能力或专业知识时；在需要承担个人责任或面对明显错误的行为时。

## 模型卡：权威影响模型

- **底层学科**：心理学、社会学
- **输入**：来自公认的、或看似为权威的专家、领导人、机构、书籍、头衔、制服等的指令或意见。
- **输出**：个体倾向于不假思索地服从和信任权威，即使在权威的建议明显错误或会导致不良后果时（如米尔格拉姆电击实验），为服从权威而放弃自己的判断。
- **失效条件**：当个体对权威的动机和能力高度怀疑时；当面临与自身核心价值观和伦理道德强烈冲突的命令时。

# 模型卡：系统思考模型

- **底层学科**：系统科学、物理学、工程学、经济学
- **输入**：一个由相互关联的组成部分（变量、人、反馈回路）组成的复杂系统；一个施加在系统中的行动或干预。
- **输出**：预测该行动不仅会产生预期的直接效果，更会产生一系列难以预测的连锁反应、次级效应和长期后果，这些后果可能削弱或颠覆行动本身的目标。系统的行为由内部互动关系决定，而非单一变量。
- **失效条件**：当系统过于简单，各部分之间的互动可以忽略时；当行动的影响范围极为有限时；当未来的反馈在时间上极度延迟，以至于改变了行动的因果关系时（补）。

# 模型卡：尺寸与极限模型

- **底层学科**：物理学、数学（缩放理论）、生物学
- **输入**：一个物体、组织或系统在尺寸、规模或时间上的变化。
- **输出**：预测随着尺寸的变化，物体的形状、功能、行为、效率以及力量/重量比等会发生非线性变化；系统存在物理或数学上的极限（临界值、瓶颈），一旦达到，系统行为会发生突变或停止。
- **失效条件**：当物体尺度极小（如纳米级别）或极大（如宇宙级别）时，经典物理学的缩放律可能不适用（补）；当人为干预（如技术突破）改变了系统的物理或数学约束时。

# 模型卡：因果逻辑模型

- **底层学科**：科学哲学、统计学、认知心理学
- **输入**：观察到的事件结果（A）；对新事件或现象的解释。
- **输出**：指出人类在理解因果关系时常犯的错误：1）原因和结果在大小上不必对应；2）多重原因常导致同一结果；3）相关性不等于因果关系；4）不能忽略基准频率和缺失数据（幸存者偏差）。该模型要求使用科学方法（假设检验、对照组、统计考量）来判断因果关系。
- **失效条件**：当因果关系是完美的确定性规律（如物理定律）时；当样本量极大且能够穷尽所有可能变量时。

---

# 模型卡：概率与风险模型

- **底层学科**：数学（概率论、统计学）、决策理论
- **输入**：一个决策或事件的可能结果及其发生的概率；损失或收益的后果严重程度。
- **输出**：1) 决策不应仅看概率，要同时考虑概率与结果（期望价值）。2) 在衡量风险时，错误的后果（尾部风险、黑天鹅）比其概率更重要。3) 人类会高估小概率（生动性）、低估大概率（熟悉性）。
- **失效条件**：当所有结果和概率都是已知的、确定的、且重复发生时（如赌博），该模型适用；当信息极度不完整，无法估计任何合理概率时，模型的应用会失效。

# 模型卡：安全边际模型

- **底层学科**：工程学、投资学、决策理论
- **输入**：对事物（资产、桥的承重、项目）价值或承载能力的最优估计；评估过程中已知的、未知的、和不可避免的不确定性。
- **输出**：为确保在面对错误、不确定性或坏运气时仍能保证安全，决策者应在估计值之上预留一个充足的缓冲区（即安全边际），牺牲一部分潜在收益来换取更高层次的安全性。
- **失效条件**：当预留的安全边际本身也基于错误的假设时；当安全边际过小，不足以覆盖极小概率但毁灭性的事件时；当追求绝对安全导致错失所有机会时。

# 模型卡：逆向思维模型

- **底层学科**：决策科学、问题解决、数学（雅各比方法）
- **输入**：一个期望的目标（如成功、幸福、盈利）或要解决的问题。
- **输出**：不是通过正向寻找成功的路径来解决问题，而是通过系统地思考“如何确保失败”（反向路径），然后积极地避开这些反向路径，从而获得成功。或者说，通过思考“如何让问题变得更糟”，来找到解决问题的关键。
- **失效条件**：当正向路径非常明确且唯一时；当问题的结构不清晰，无法反向构建路径时；过度使用可能导致过度悲观和消极（补）。

# 模型卡：多学科心智模型格栅模型

- **底层学科**：认知科学、教育心理学
- **输入**：一个跨学科的、需要从多角度理解的复杂问题。
- **输出**：为了解决单一学科无法解决的问题，个体应建立一个由心理学、物理学、生物学、经济学、数学等多个学科的核心“模型”组成的知识格栅。将这些模型综合运用，可以更全面、更准确地理解世界，避免仅凭单一学科（如“锤子综合症”）导致的错误判断。
- **失效条件**：当个体学习的模型本身不正确或不完整时；当模型之间相互矛盾或无法有效整合时；当面临完全崭新的、现有模型无法解释的现象时。

# 模型卡：证据评估模型（贝叶斯推理的应用）

- **底层学科**：统计学、科学哲学、认知心理学
- **输入**：1) 某个主张或假设的先验概率（基于背景知识或基准频率）。2) 新的证据（如目击证词、DNA匹配、检验结果、专家意见）。
- **输出**：个体的最终判断（后验概率）应基于先验概率与证据的可靠性（如敏感度、特异性、随机匹配概率）的综合计算。必须警惕案例证据（生动性）对基于统计的基准频率的破坏，要区分“在”与“看到”之间的逻辑差异。
- **失效条件**：当无法获得任何有代表性的基准频率或先验概率时；当证据的可靠性（如假阳性率、随机匹配率）完全未知时；当个体情绪激动，无法进行逻辑计算时（补）。

# 模型卡：回归平均值效应模型

- **底层学科**：统计学
- **输入**：一系列包含随机因素的事件结果中的极端表现（如特别高或特别低的分值、业绩、运动表现）。
- **输出**：在纯粹由机会或部分随机因素决定的系列中，极端表现之后的结果，在统计上会倾向于向该系列的平均值靠拢。若忽略此效应，容易将这种自然回归错误地归因于某项干预或态度变化。
- **失效条件**：当全部结果都是由系统性、非随机因素决定的，且不存在测量误差时；当极端值之后出现的系统性干预或条件发生了根本性、永久性的改变，使得平均值本身发生了移动。
